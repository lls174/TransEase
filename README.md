# TransEase - AI翻译助手

一款基于智谱AI GLM-4-Flash模型的Chrome翻译插件，支持选中文字自动翻译，自动识别源语言，多语言翻译。

## ✨ 功能特点

- **自动识别源语言** - AI自动检测原文语言，无需手动选择
- **多语言支持** - 支持13种语言：中文、英语、日语、韩语、法语、德语、西班牙语、俄语、葡萄牙语、意大利语、阿拉伯语、泰语、越南语
- **实时切换目标语言** - 可在翻译窗口中即时切换目标语言
- **一键复制** - 快速复制翻译结果
- **国内可用** - 使用智谱AI API，国内网络可正常访问
- **免费使用** - 基于GLM-4-Flash免费模型

## 📦 安装步骤

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `TransEase` 文件夹

## ⚙️ 配置说明

1. 点击插件图标打开设置页面
2. 输入智谱AI API密钥（访问 <https://open.bigmodel.cn/> 免费注册获取）
3. 选择默认翻译语言
4. 点击保存设置

## 💡 使用方法

1. 在任意网页中选中需要翻译的文字
2. 右键点击选中的文字，在弹出的菜单中选择 "TransEase 翻译"
3. 可在窗口顶部切换目标语言
4. 点击「复制译文」按钮复制翻译结果
5. 点击右上角关闭按钮或点击页面其他地方关闭窗口

## 🛠 技术实现

- **开发环境**：Chrome扩展开发
- **AI模型**：智谱AI GLM-4-Flash
- **接口**：智谱AI API v4
- **权限**：storage, activeTab, contextMenus
- **语言**：JavaScript, HTML, CSS

## 📁 项目结构

```
TransEase/
├── manifest.json    # Chrome扩展配置文件
├── background.js    # 后台服务脚本（处理翻译API调用）
├── content.js       # 内容脚本（页面交互逻辑）
├── popup.html       # 设置弹窗页面
├── popup.js         # 设置弹窗脚本
├── styles.css       # 翻译弹窗样式
└── icon.png         # 插件图标
```

## 🔧 API配置

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册并登录账号
3. 进入「API密钥管理」页面
4. 生成新的API密钥
5. 复制密钥到插件设置页面

## 📝 注意事项

- 每次调用API会消耗相应的token，GLM-4-Flash有免费额度
- 请合理使用API，避免频繁请求导致API被限制
- 翻译结果仅供参考，可能存在误差
- 使用AI翻译，请注意数据安全
